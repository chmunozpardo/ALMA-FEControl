Detailed Description

namespace for all items relating to Front End configuration

The following band-specific tuning behaviors are supported: band3: 2 pols, 2 SIS/pol, no magnets.

SIS voltage (VJ) and LO PA drain voltage (VD) are optimized for max output power and lowest noise. band6: 2 pols, 2 SIS/pol, one magnet coil/pol controls both junctions (IMag01, IMag11). LO PA (VD) is adjusted to set target SIS current (IJ). band7: 2 pols, 2 SIS/pol, separate magnets on all junctions. Magnet current is optimized to minimize junction Josephson current with VJ=0. LO PA (VD) is adjusted to set target SIS current (IJ). band9: 2 pols, 1 SIS/pol, one magnet pol (IMag01, IMag11). Magnet current is optimized to minimize Josephson current with VJ swept +/- 0.2 mV. Output power is measured while sweeping VJ. LO PA (VD) is adjusted to set target SIS current (IJ).
