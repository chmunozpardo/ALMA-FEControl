Detailed Description

The class representing a generic cartridge assembly, including cartrioperations for which a CartAssembly is responsible.

All cartridge operations are implemented here or in the constituent ColdCartImpl or WCAImpl objects. Cartridge behavior which varies from band-to-band is discriminated here.
