bool MixerHeating::start ( int  pol,
float  targetTemp,
int  timeout 
)

start the mixer heating process.

If pol == -1, heat both polarizations. Else pol must be 0 or 1. If targetTemp >= 4, retrigger the the heater for up to timeout seconds to attain the target temperature. Else, retrigger the heater for exactly timeout seconds. For band 9, timeout may be no longer than 2 seconds. For bands 678, timeout may be up to 2 minutes.
